背景与现状

现有数据集的核心问题

结构性缺陷

  • 缺乏科学的逻辑结构划分
  • 美学因子之间存在冗余
  • 标注覆盖范围有限
  • 评分标准跨数据集不统一

方法论挑战

  • 主观性强，评分者间差异大
  • 文化/个体偏好影响标注质量
  • 单一MOS分数难以表达多维美学
  • 缺少国际标准指导构建

ITU-T 标准化突破

2024年，ITU-T 发布首个图像美学评估国际标准（ITU-T H.Sup.IAA），从摄影师、相机、观看者三个角色维度定义美学属性体系，填补了移动终端计算摄影美学评估的标准空白。

ITU 三角色模型

摄影师维度（拍摄角度、构图技巧）→ 相机维度（曝光度、设备属性）→ 观看者维度（主观偏好评价）。三层递进，从客观技术到主观感受。

评测指标体系

Benchmark 评价指标

回归类指标

  • SRCC — Spearman秩相关系数
  • PLCC — Pearson线性相关系数
  • KRCC — Kendall秩相关系数
  • MSE / MAE — 均方/绝对误差

分类类指标

  • Accuracy — 等级分类准确率
  • Precision / Recall / F1
  • 感知准确率（QA三类题型）
  • 描述质量（完整性/精确性/相关性）

UNIAA 三级评测

Perception（感知QA）→ Description（美学描述）→ Assessment（零样本评分），递进式全面覆盖。

HumanAesExpert 多头融合

LM Head + Regression Head + Expert Head → MetaVoter 加权聚合，兼顾分类与回归。

关键启示

三种范式各有优劣：社区众包成本低但标注噪声大；专家标注质量高但规模受限；多源融合需解决标准不统一问题——IDCP 范式提供了一条中间路径。

总结与建议

构建高质量评测集的核心要素

维度完整性

覆盖内容/构图/色彩/光线/焦点/情感六维，并区分三角色（拍摄者/设备/观者）层次。

标注严谨性

≥9人MOS、迭代培训筛选、5级量表、跨文化分布控制、离群剔除。

数据多样性

自然图/艺术图/AI生成图混合，场景与质量梯度均衡，避免分布偏置。

评测全面性

感知→描述→评分三级递进，回归+分类双轨指标，对齐人类表现基线。

推荐实践路径

遵循 ITU-T 标准定义维度 → 参考 UNIAA IDCP 范式高效转换已有数据 → 采用 HumanAesExpert 迭代标注协议确保质量 → 建立三级评测体系全面验证模型能力。
